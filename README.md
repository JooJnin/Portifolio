### O projeto foi desenvolvido utilizando as seguintes tecnologias:

  - HTML5: Estruturação de conteúdo.

  - CSS3: Estilização e responsividade.

  - JavaScript: Lógica e interatividade.

  - Figma: Utilizado para a criação do protótipo e design de interface (UI/UX).

  - GitHub: Plataforma utilizada para versionamento de código e hospedagem do repositório.

## Design (Prototipagem)

O design da interface foi planejado no Figma para garantir a melhor experiência de usuário antes da implementação do código.

 - Você pode visualizar o protótipo clicando no link abaixo: [[Link para o projeto no Figma](https://www.figma.com/design/HfgVVQkvSzFqzwj6C6VY3x/Portf%C3%B3lio?node-id=11-24&t=WFB0ZFCqtELqIoTT-1)]


## Versionamento

 - Para o controle de versões, foi utilizado o Git integrado ao GitHub, seguindo boas práticas de commits e organização de branches.
